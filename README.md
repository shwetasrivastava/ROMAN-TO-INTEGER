# Roman To Integer Converter
<br/>

## Package to convert roman numbers to integer equivalent.

Example:
```
 I : 1
 II : 2
 III : 3
 IV : 4
 V : 5
 LVIII : 58
```

## How to use?
- Install package from npm
```
npm i roman-to-integer --save
```

##USAGE

var romanToInteger = require('roman-to-integer');

romanToInteger('I');

##OUTPUTS

1

##KEYWORDS

Roman numbers to integer
Roman
roman
integer
